संस्कृत -हिन्दी अनुवादकम् अनुसारकम् च
(Sanskrit-Hindi Accessor Cum Machine Translator) Release 0.66 (Feb. 8, 2011)
 

A Sanskrit-Hindi Accessor cum Machine Translator is a device that
a) produces Hindi translation of a Sanskrit text, and also
b) provides intermediate outputs at various levels viz. पद-विश्लेषण, कारक-विश्लेषण, शब्दार्थ-निर्धारण, आदि .



First Step

The present SHMT system works on Linux operating system. The current version of SHMT tested on UBUNTU operating system. We recommoned the UBUNTU 10.04+ operating system. Before installing SHMT system, install all the following softwares/packages on your machine. The softwares/packages can be installed from System -> Administration -> Synaptic Package Manager directly.

  1. g++
  2. flex
  3. automake
  4. libxml2-2.26+
  5. lttoolbox-3.1+
  6. apertium3+
  7. apache2 [To enable public_html, the instructions with images are given below in Second Step.]
  8. bash
  9. Perl (v5)
  10. Python (v2.0+)
  11. gdbm library
  12. Graphviz (v2.20.2+)


Second Step:

In this step, We will see how to enable public_html. Please follow the instructions which are given below:

  • Open Terminal from Application -> Accessories -> Terminel



  • Type the following commands into bash:

    sudo mv /etc/apache2/mods-available/userdir.* /etc/apache2/mods-enabled/

    It may ask the password for authentication. Just enter the password. See the screenshot below.




  • Open the mime.conf file by typing the following commands into bash: sudo gedit /ete/apache2/mods-enabled/mime.conf and remove # symbol from begining of the following line #AddHandler cgi-script .cgi and also add the following newline AddHandler cgi-script .pl . See the screenshot below.






  • Open the userdir.conf file by typing the following commands into bash: sudo gedit /etc/apache2/mods-enabled/userdir.conf and find IncludesNoExec text and replace it with ExecCGI See the screenshot below.






  • Now, to activate public_html support. Restart apache server by typing following commands into bash sudo /etc/init.d/apache2 restart See the pictures below.







Third Step:


How to install:

  • Open SHMT/MT/Makefile in an editor
  • Set the values of APACHE_RUN_USER and APACHE_RUN_GROUP for your linux distribution.
  • Save and exit.
  • Run the following commands:-
    • make
    • sudo make install

How to Run:

in Web mode:

in firefox: http://localhost/~userid/scl/SHMT/shmt.html

in batch mode:

mkdir $HOME/public_html/scl/SHMT/MT/prog/test

create a file in WX notation in this directory

../shell/anu_skt_hnd_batch.sh <file>

o/p is in <file>_frame.html

<file>_translate.html contains the translation.

<file>.html contains the intermediate o/ps.
